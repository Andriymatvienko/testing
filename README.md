# Консольний застосунок «Нотатник» (Console Notebook App)

Проєкт представляє собою легкий та інтуїтивно зрозумілий консольний менеджер нотаток, розроблений мовою **Python**. Застосунок дозволяє користувачам ефективно створювати, переглядати, шукати, редагувати та видаляти текстові замітки у зручному інтерактивному меню. 

Даний проєкт також використовується у навчальних цілях для демонстрації практик тестування програмного забезпечення (Unit Testing) за допомогою фреймворків `unittest` та `pytest`.

---

## Основний функціонал

Програма підтримує повний життєвий цикл управління даними (CRUD) через консольний інтерфейс:

1. **Додавання нотаток:** Створення нових записів із автоматичною генерацією унікального `ID` та фіксацією точного часу створення.
2. **Перегляд усіх нотаток:** Відображення списку з можливістю динамічного сортування (за алфавітом, часом створення тощо).
3. **Регістронезалежний пошук:** Швидке знаходження інформації за ключовими словами як у заголовках, так і всередині тексту нотаток.
4. **Модифікація даних:** Безпечне редагування та повне видалення записів за їхнім ідентифікатором.
5. **Персистентне збереження даних:** Автоматичний експорт та імпорт даних у локальний файл формату `JSON`.

---

### Головне меню програми
При запуску застосунку користувача зустрічає структуроване інтерактивне меню з валідацією введення:

```text
========================================
        НОТАТНИК - ГОЛОВНЕ МЕНЮ
========================================
  1. Додати нотатку
  2. Переглянути всі нотатки
  3. Пошук нотаток
  4. Редагувати нотатку
  5. Видалити нотатку
  0. Вийти
========================================
Ваш вибір:
```
## Системні вимоги та встановлення
Для запуску проєкту на вашому локальному комп'ютері мають бути встановлені:
- Python версії 3.10 або вище.
- Менеджер пакетів pip (зазвичай постачається разом із Python).

### Кроки для запуску:
1. Клонуйте репозиторій:
```bash
git clone [https://github.com/Andriymatvienko/testing](https://github.com/Andriymatvienko/testing)
cd testing
```
2. Створіть та активуйте віртуальне оточення (рекомендовано):
```bash
python -m venv venv
# Для Windows:
venv\Scripts\activate
# Для macOS/Linux:
source venv/bin/activate
```
3. Встановіть залежності для тестування:
```bash
pip install pytest
```
4. Запустіть застосунок:
```bash
python main.py
```
### Автоматизоване тестування
Проєкт повністю покритий модульними тестами. Для перевірки працездатності логіки та відсутності регресій запустіть pytest:
```bash
pytest test_notebook_pytest.py -v
```
